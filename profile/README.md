## 매지구름 프로젝트

일기와 함께 **나의 감정 상태**를 기록하고, 친구와 공유해보세요!   
매일 일기를 작성하면 나의 **월별 감정 상태**를 확인할 수 있어요!  
**피드 기능**으로 나와 가장 친한 친구에게 나만의 이야기를 들려주세요! 내가 모르던 친구의 속마음을 듣고 공유해보세요!
